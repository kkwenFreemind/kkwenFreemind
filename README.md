- 🤖 **智能自然語言查詢平台（AI Interface System）**：  
  - 基於 Spring Boot 的智能查詢系統，整合 Google Gemini AI，自動將中文自然語言轉為 PostgreSQL 查詢語句  
  - 支援圖表類型推薦（圓餅圖、長條圖、折線圖、散佈圖、表格），並透過意圖分析與資料結構智能選擇最佳展示方式  
  - 完整企業級權限體系，包括使用者角色管理、動態選單控制、Redis 快取、SQL 安全驗證機制（僅允許 SELECT、JSQLParser 防注入）  

- 📂 **文件檢索問答系統（RAG System）**：  
  - 透過 **向量資料庫 (ChromaDB)** 與 **Google Gemini**，將文件內容轉換為可檢索的知識庫  
  - 使用者可上傳 PDF、TXT、MD 檔案，系統自動切分並建立嵌入向量，支援語意檢索與問答  
  - 完整串接 FastAPI + PostgreSQL，支援 Token 認證、檔案上傳、索引管理與查詢結果回應  
  - 展現 **RAG（Retrieval-Augmented Generation）架構** 在實務應用的落地能力，適合知識管理與企業內部 FAQ 系統
